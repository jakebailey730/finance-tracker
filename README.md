# Finance Tracker

A personal finance tracking web application designed to help users monitor spending, manage transactions, and analyse financial activity through a clean and responsive interface.

## Features

* Transaction tracking
* Expense categorisation
* Dashboard summaries
* Responsive UI
* Financial data management

## Technologies Used

* PHP
* HTML/CSS
* Bootstrap
* JavaScript
* JSON / Database

## Goals

This project is being used to strengthen understanding of:

* Full-stack web development
* Backend application structure
* Data handling and persistence
* Responsive UI design
* Maintainable software development practices
